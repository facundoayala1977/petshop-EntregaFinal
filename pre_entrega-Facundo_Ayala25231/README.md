Este proyecto es una página web básica desarrollada como parte de un curso de Front-End. La página está estructurada con HTML semántico y utiliza las etiquetas: `<header>`, `<nav>`, `<main>` y, `<footer>` para organizar el contenido. El objetivo es aprender a crear la estructura básica de una página web con estilo de CSS y en un futuro, JavaScript. Tambien cuenta con un formulario de contacto en formspree, con los siguientes campos: Nombre, Correo electronico, Mensaje y boton de Enviar. Todos los campos estan configurados como obligatorios.

La página Peluditos Pet Shop es de un local dedicado a la venta de alimentos, accesorios y productos para mascotas ubicado en Ituzaingó, Corrientes.

---

Estructura del proyecto:

index.html → Estructura principal del sitio.  
/css/style.css → Estilos del sitio con Flexbox y Grid.  
/imagenes/ → Logo, íconos de redes y productos.  
/puppy_dog_pals.mp4 / .vtt → Video de presentación con subtítulos. (Nota: habia descargado otro video con subtítulo pero tuve problemas al colocar el enlace)

---

Contenido principal

Cuerpo Principal - <body> donde se encuentran <header>, <main>, <footer>

Encabezado - <header>
-Incluye el logo, el título principal y la barra de navegación interna.

Presentación
Video con subtítulos personalizados y póster inicial.

Productos
Catálogo de productos ofrecidos, los cuales se encuentran distribuidos en tarjetas con imagen, descripción y botón de compra.

Reseñas
Sección con tres comentarios de clientes, organizada con CSS Grid.

Ubicación y multimedia
Mapa embebido de Google Maps y video informativo de YouTube.

Formulario de contacto
Formulario funcional mediante Formspree, con campos validados y diseño responsivo.

Footer
Incluye enlaces rápidos, dirección del local y redes sociales con íconos.

---

Tecnologías utilizadas

-HTML5 para la estructura.  
-CSS3 (Flexbox y Grid) para el diseño, agregando media queries para hacer responsivo.  
-Formspree para el envío de formularios.  
-Google Fonts para la tipografía.  
-YouTube y Google Maps embeds para multimedia.  

---

Uso

1. Clonar o descargar el repositorio.  
2. Mantener la estructura de carpetas (`css`, `imagenes`, `videos`).  
3. Abrir `index.html` en el navegador.
